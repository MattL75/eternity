xPowerY
xPowerY(x, y, rounds)

This function calculates x^y for x > 0 using the natural log (ln) series as follows: ln(x) = 2[ ((x - 1) / (x + 1)) + (((x - 1) / (x + 1)) ^ 3) / 3 + (((x - 1) / (x + 1)) ^ 5) / 5 + ...]

Parameters :
